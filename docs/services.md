# 服务范围

当前主打方向是回国网络，重点关注低延迟、大带宽和长期稳定使用体验，同时支持 `RouterOS` 和星链回国场景。

## 1. 回国网络与软路由搭建

可承接以下方向的基础搭建与整理：

- 回国网络环境
- `RouterOS`
- `OpenWrt`
- `iKuai`
- Linux 软路由
- 星链回国网络场景
- 家用轻量网络环境
- 小团队或工作室网络环境

可做内容包括：

- 新环境初始化
- 基础网络结构整理
- 网关、DNS、DHCP 等基础配置
- 分流和访问策略整理
- 常见转发与 NAT 配置
- 回国访问相关的策略整理
- 面向低延迟和大带宽目标的基础优化
- 星链回国接入后的基础网络整理

## 2. 线路维护

适合已经有现成网络环境，但长期存在以下问题的客户：

- 线路不稳定
- 延迟偏高
- 带宽利用率不理想
- 高峰时段体验差
- 访问策略混乱
- 配置长期无人维护
- 出故障时难以定位问题

可提供：

- 日常巡检
- 问题复盘
- 配置调整
- 结构梳理
- 维护建议
- 线路质量跟踪与优化建议

## 3. 故障排查

常见排查范围：

- 路由异常
- DNS 异常
- 访问慢或不通
- 回国访问延迟高
- 带宽跑不满
- 丢包和波动明显
- 分流策略失效
- 局域网设备访问异常
- 端口转发不生效

## 4. 长期维护

如果你不只是一次性搭建，而是希望有人长期维护，也可以按长期协作方式处理：

- 定期检查
- 配置调整
- 新需求接入
- 设备变更后的网络迁移
- 出现异常时协助恢复
- 持续优化回国线路体验

## 5. 适合合作的场景

- 海外回国网络使用场景
- 星链回国使用场景
- 个人家庭网络
- 内容工作室
- 小规模办公环境
- 需要持续维护的软路由环境

## 6. 不主打的方向

当前仓库不主打以下类型的公开宣传：

- 大型企业集成项目
- 纯硬件销售
- 与实际交付无关的空泛咨询
